# Glossary

Date: March 11, 2026

## Terms

- Airline Administrator: the user responsible for setting up planes, flight classes, flights, and manual reallocation actions.
- Customer: the user who searches flights and manages reservations.
- Plane: an aircraft in the airline fleet used to operate scheduled flights.
- Flight: a planned travel service between an origin and a destination on a specific date.
- Flight Class: a travel class offered on a flight, such as First Class, Coach, or Economy.
- Reservation: a booking made by a customer for a flight and a selected class.
- Unconfirmed Reservation: a reservation that has been created but not yet confirmed through payment.
- Confirmed Reservation: a reservation whose payment has been confirmed.
- Cancelled Reservation: a reservation that has been cancelled and is no longer active.
- Refund: money returned to the customer after a valid cancellation.
- Refund Policy: the rule used to determine whether and how much money is refunded after cancellation.
- Reallocation: a manual action that removes a seat from an older unconfirmed reservation when seats are scarce.
- Payment Confirmation: the system action that changes a reservation from unconfirmed to confirmed.
- Seat Inventory: the available number of seats for a given flight class on a flight.
- Origin: the departure location of a flight.
- Destination: the arrival location of a flight.
